1. Layer Selection

ලේයරයෙන් කොටසක් Select කර එය වෙන් කර ගැනීමට හෝ Delete කිරීමට Marquee Tool, Polygonal Tool හෝ Pen Tool යොදා ගත හැක. පහතින් දැක්වෙන්නේ පිළීවෙලින් පහළට ඒවා ඇති ස්ථාන සහ කෙටි මං යතුරු.

  1. Marquee Tool - M
  2. Polygonal Tool - L
  3. Pen Tool - P

Marquee Tool මගින් චතුරස්‍ර හැඩයක් ලබා ගත හැකි අතර Polygonal Tool හා Pen Tool මගින් ඇඳීමෙන් අවශ්‍ය හැඩයක් ලබා ගත හැක.


2. Polygonal Tool

Polygonal Tool මගින් ඇඳීමේ දී ආරම්භ ස්ථානය නැවත යා කිරීමෙන් Select වීම සිදු වේ. එසේ නොමැතිනම් අවසන් කළ ස්ථානයේ මූසිකය Double Click කර විට ආරම්භ ස්ථානය සමග යා වී Select වීම සිදු වේ.


3. Pen Tool

Pen Tool මගින් ඇඳීමට පෙර Pen Tool එක ගෙන Tool Mode එක Path ලෙස සකසන්න.

මෙමගින් ඇඳීමේ දී වක්‍ර ආකාරයේ ඇඳීම් කළ හැක.

එමෙන්ම වක්‍රයක් ඇඳි පසු ඇඳීමේදී එය වක්‍ර වෙනාවන රේඛාවක් විය යුතුනම් පෙර ඇඳි තිත මත Alt යතුර තද කරගෙන Click කර වක්‍ර බව නැති කළ යුතුය.

ඇඳගත් කොටස Selection එකක් කිරීම Pen Tool හිදී කළ යුතුය. මේ සඳහා Ctrl + Enter ඔබන්න. එමෙන්ම ආරම්භක හා අවසාන ස්ථාන යා වීම මෙහිදී සිදු වේ.


4. Copy Selection

ඇඳගත් කොටස පිටපත් කර ගැනීමට Ctrl + J ඔබන්න. මෙහිදී Layer Pallert එකේ තෝරා ඇති ලේයරය ගැන සැළකිළිමත් වන්න. මක් නිසාදයත් පිටපත් වීම සිදු වන්නේ තෝරා ඇති ලේයරයෙන් බැවිණි.


5. Cut Selection

ඇඳගත් කොටස වෙන් කර ගැනීමට Ctrl + Shift + J ඔබන්න. මෙහිදී Layer Pallert එකේ තෝරා ඇති ලේයරය ගැන සැළකිළිමත් වන්න. මක් නිසාදයත් වෙන්වීම වීම සිදු වන්නේ තෝරා ඇති ලේයරයෙන් බැවිණි.


6. Delete Selection

ඇඳගත් කොටස මකා දැමීමට යතුරු පුවරුවේ Delete යතුර ඔබන්න. මෙහිදී Layer Pallert එකේ තෝරා ඇති ලේයරය ගැන සැළකිළිමත් වන්න. එහි දී Delete විය යුතු ආකාරය වෙනස සැකසීමට කවුළුවක් ලැබේ.

එහි Contents යන්න Background Color ලෙස තෝරා OK කිරීමෙන් පහත ලෙස මකා දැමේ.


7. Deselect

Select වූ කොටසේ Path එක මැකීම සඳහා Ctrl + D ඔබන්න.
