1. Clone Stamp Tool

ලේයරයක යම් කොටසක් තව තැනකට පිටපත් කිරීම මෙහිදී කළ හැක. මෙය වැඩි වශයෙන් භාවිතා කරන්නේ අනවශ්‍ය කොටස් සැඟවීමටයි. Clone Stamp Tool එක Tool Menu එකෙන් ලබා ගත හැකි අතර S යතුර එබීමෙන් ද ලබා ගත හැක.

අවශ්‍ය ලේයරය තෝරා ගන්න. දැන් මුලින්ම කළ යුත්තේ Alt යතුර සමග යම් තැනක් Click කිරීමයි. පසුව Alt යතුර අත හැර ඇඳීමට අවශ්‍ය තැන ඇඳීමෙන් පෙර තෝරාගත් තැනේ පසුබිම නව තැනට පිටපත් වේ.


2. Eraser Tool

ලේයරයක යම් කොටසක් මකා දැමීමට මෙය භාවිතා වේ. Eraser Tool එක Tool Menu එකෙන් ලබා ගත හැකි අතර E යතුර එබීමෙන් ද ලබා ගත හැක.

අවශ්‍ය ලේයරය තෝරාගෙන මැකීම සිදු කළ හැක.

එමෙන්ම මෙම Tool එක ගෙත් පසු ඉහළින් දැක්වෙන සැකසීම් ද සිදු කිරීමෙන් මකන ආකාරය වෙනස් කළ හැක.


3. Blur Tool

ලේයරයක යම් කොටසක් තව බොඳ කිරීමට මෙය බාවිතා වේ. Tool Menu එකෙහි මෙම Tool එක ඇති අතර කෙටි මං යතුරක් නොමැත.

එමෙන්ම මෙම Tool එක ගෙත් පසු ඉහළින් දැක්වෙන සැකසීම් ද සිදු කිරීමෙන් බොඳ කරන ආකාරය වෙනස් කළ හැක.

අවශ්‍ය ලේයරය තෝරා ගත් පසු බොඳ කිරීම සිදු හැක.
