1. Create New Document

නව ගොනුවක් නිර්මාණය කිරීමට File Menu එකෙහි New... අයිතමය තෝරන්න. මේ සඳහා Ctrl + N යන කෙටි මං යතුර ද භාවිතා කළ හැක.

එවිට දිස් වන Window එකෙහි අදාළ සැකසුම් කළ හැක.

OK විධානය ලබා දුන් විට නව ගොනුවක් නිර්මාණය වේ.


2. Open Document

කලින් නිර්මාණය කළ හෝ, වෙනත් ඡායාරූපයක් Open කර ගැනීමට File Menu එකෙහි Open... අයිතමය තෝරන්න. Ctrl + O යන කෙටි මං යතුර මගින්ද මෙය සිදු කළ හැක.

එවිට දිස් වන Open File කවුළුවෙන් අදාළ ගොනුව තෝරන්න. එම ගොනුව මත Double Click කිරීම හෝ එය මත Click කර Open මත Click කිරීම හෝ කළ යුතුය.


3. Document Tabs

විවෘත කරන ලද ගොනු කිහිපයක් ඇති විට ඒවා Tab Menu ආකාරයෙන් ඉහළින් බලා ගත හැක. අවශ්‍ය ගොනුවේ ටැබ් එක මත Click කිරීමෙන් එය දිස් වේ.

එම Tab එකන් අල්වා පහතට ඇදීමෙන් ඒවා වෙන් වශයෙන් දැක ගත හැක.
