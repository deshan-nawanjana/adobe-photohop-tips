1. Move Layer

Layers Move කිරීම සඳහා Move Tool එක ලබා ගත යුතුය. එය Tool Menu එකෙහි Move Tool ලෙස තිබේ. නැතිනම් යතුරු පුවරුවෙහි V යතුර ඔබා එය ලබා ගත හැක‍.

ඉහලින් ඇති Auto Select යන්න සක්‍රීය විට මූසිකය මගින් තෝරන Layer එක පහසුවෙන් එහා මෙහා කළ හැක.

එසේ නොමැති විට Layer කොටසෙහි අවශ්‍ය ලේයරය Select කර තිබිය යුතුය.


2. Copy Layer

ලේයරයක් පිටපත් කර ගැනීමට Ctrl + J කෙටි මං යතුර භාවිතා කළ හැක. ඒවිට Select කර ඇති ලේයරයේ පිටපතක් ලැබේ.


3. Layers Order

ඡායාරූපයේ Layers අනු පිළිවෙළ Layer Pallert එකෙන් බලා ගත හැක. අවශ්‍ය Layer එක මූසිකයෙන් අල්ලා ඉහළ පහළ කිරීමෙන් අවශ්‍ය අනුපිළිවෙළ සැකසිය හැක.

එමෙන්ම Select කර ඇති Layer එක ඉහළට ගෙන ඒමේ කෙටි මං යතුර Ctrl + ] වේ.
Ctrl + Shift + ] මගින් අදාළ ලේයරය එක වරම ඉහළටම පැමිණේ.

Select කර ඇති Layer එක පහළට ගෙන යාමට කෙටි මං යතුර Ctrl + [ වේ.
Ctrl + Shift + [ මගින් අදාළ ලේයරය එක වරම පහළටම යැවේ.


4. Hide Layers

අවශ්‍ය නොවන නමුත් පසුවට අවශ්‍ය වේ යැයි හැඟෙන ලේයර Hide කර තැබීමට Layer Pallert එකෙහි අදාළ ලේයරය ඉදිරියෙන් ඇති ඇසක ලකුණ මත Click කරන්න. නැවත ඒ මත Click කළ විට සැඟවුණු ලේයරය නැවත ලබා ගත හැක.


5. Delete Layers

Move Tool එක මගින් ලේයරය තෝරා යතුරු පුවරුවේ Delete යතුර එබීමෙන් Layer එක Delete කළ හැක. එසේ නැතිනම් Layer Pallert එකෙහි අදාළ ලේයරය Click කර යතුරු පුවරුවේ Delete යතුර එබීමෙන් ද Layer එක Delete වේ.
