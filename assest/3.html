2. Canvas Size

මෙමගින් ඡායාරූපයේ වෙනසක් සිදු නොවන අතර ඡායාරූපය ඇති වට ප්‍රමාණය වෙනස් කිරීමක් සිදු වේ. මේ සඳහා File Menu එකෙහි Image > Canvas Size... තෝරන්න. මේ සඳහා කෙටි මං යතුර වන්නේ Ctrl + Alt + C

එවිට දිස් වන Canvas Size කවුළුවෙහි Width සහ Height මගින් Canvas එකෙහි විශාලත්වය ද, Anchor මගින් විශාලනය විය යුතු දිශාව ද ලබා දිය හැක. එමෙන්ම Canvas Extexnsion color යනු නව ඉඩ ප්‍රමාණයට ලැබිය යුතු පසු බිම් වර්ණයයි.


3. Crop Image

යම් ඡායාරූපයක කොටසක් කපා ගැනීම මෙමගින් කළ හැක. Tools මෙනුවෙහි Crop Tool මගින් ද, යතුරු පුවරුවේ C යතුර මගින් ද මෙම Tool එක ලබා ගත හැක.

පසුව අදාළ කොටස Select කර Enter යතුර තද කිරීමෙන් Crop වීම සිදු වේ. එමෙන්ම Shift යතුර තද කරගෙන Select කිරීම මගින් සම චතුරස්‍රාකාරයටද, Alt යතුර තද කරගෙන Select කිරීම මගින් Select කිරීම ආරම්භ කළ ස්ථානයේ සිට වටේට විහිදෙන ලෙස ද Select කළ හැක.
